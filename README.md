# del-console README

use this plugin, you can del console in your project

## del console in file
### use command 
    del-console-file
### use shortcut key  
    win：cmd+shift+f
    mac：cmd+shift+f

## del console in folder(src/*)
### use command 
    del-console-folder
### use shortcut key  
    win：cmd+shift+m
